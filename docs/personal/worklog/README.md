# 작업일지

## 구조

```
worklog/
├── README.md
├── {year}/
│   ├── {mm}/
│   │   └── {mm}-{dd}.md    # 일일 작업일지
│   ├── W{nn}.md             # 주간 요약
│   ├── {mm}.md              # 월간 요약
│   └── {year}.md            # 연간 요약
```

## 운영 규칙

| 주기 | 작성 시점 | 방법 |
|------|----------|------|
| 일일 | `/wrap` 실행 시 자동 생성 | 세션 리포트 기반 |
| 주간 | 금요일 또는 다음주 월요일 | 해당 주 일일 로그 집계 |
| 월간 | 월말 또는 다음달 초 | 해당 월 주간 요약 집계 |
| 연간 | 연말 | 월간 요약 집계 |

## 일일 로그 템플릿

```markdown
# YYYY-MM-DD 작업일지

## 완료
- ...

## 진행 중
- ...

## Jira 이슈
| 이슈 | 작업 내용 | 상태 변경 |
|------|----------|----------|

## 코드 리뷰
- ...

## 특이사항
- ...
```

## /wrap 연동

`/wrap` 실행 시 세션 리포트의 "완료된 작업" 섹션을 기반으로 해당일 작업일지를 자동 생성/갱신합니다.
