# Luppiter Web - 월보드 상세 화면 구조

## 개요

본 문서는 Luppiter Web의 **월보드 상세** 화면의 구조와 처리 흐름을 설명합니다.

---

## 화면 정보

- **URL**: `/dashboard/specific`
- **JSP 파일**: `web/dashboard/specificMain.jsp`
- **Controller**: `DashboardController.java`
- **설명**: 월보드 상세 - 전체/공공/민간/Captive

---

## 화면 구성 요소

### 1. 헤더 영역
- **실시간 배지**: `horizontal-badge` - 현재 시간 표시

### 2. 슬라이더 영역
- **대형 슬라이더** (`slider-wrap large`):
  - 전체/공공/민간/Captive 선택
  - 네비게이션 버튼 (좌/우)
- **서브 슬라이더** (`slider-wrap small`):
  - Zone별 선택
  - 재생/일시정지 버튼

### 3. 카드 영역 (`card-wrap card-5`)
- **Zone별 카드**: `zoneTypeArea`
  - Zone별 관제 정보 표시

---

## 처리 흐름

### 1단계: 화면 초기화

```
[페이지 로드]
    │
    └─► DashboardController.viewSpecificMain()
        ├─► 공통코드 주입 (DASHBOARD_CENTER_INCIDENT_STATUS)
        └─► specificMain.jsp 렌더링

    ▼

[JavaScript 초기화]
    │
    └─► SpecificMainDashboard.js 로드
        ├─► 슬라이더 초기화
        └─► 데이터 조회 시작
```

#### 주요 코드 위치
- **컨트롤러**: `DashboardController.java:134-147`
- **JSP**: `web/dashboard/specificMain.jsp`

### 2단계: 데이터 조회

```
[IDC 정보 조회]
    │
    └─► POST /dashboard/specific/IDC/info
        ├─► controlType 파라미터 (전체/공공/민간/Captive)
        └─► IDC 목록 조회

    ▼

[Zone 정보 조회]
    │
    └─► POST /dashboard/specific/zone/info
        ├─► zoneType 파라미터
        └─► Zone별 관제 기본 정보 조회

    ▼

[Zone 상태 조회]
    │
    └─► POST /dashboard/specific/zone/status
        ├─► zoneType 파라미터
        └─► Zone별 관제 상태 정보 조회
```

#### 주요 코드 위치
- **API 엔드포인트**: `DashboardController.java:148-234`

## 주요 API 엔드포인트

| API | 메서드 | 엔드포인트 | 설명 |
|-----|--------|------------|------|
| IDC 정보 조회 | POST | `/dashboard/specific/IDC/info` | IDC 목록 조회 |
| Zone 정보 조회 | POST | `/dashboard/specific/zone/info` | Zone별 관제 기본 정보 조회 |
| Zone 상태 조회 | POST | `/dashboard/specific/zone/status` | Zone별 관제 상태 정보 조회 |

## 필요한 데이터 (API 요청)

|--------|------|------|-----------|
| `controlType` | String | 관제 타입 (전체/공공/민간/Captive) | 필수 |

### Zone 정보 조회 파라미터

| 필드명 | 타입 | 설명 | 필수 여부 |

## 데이터 흐름 다이어그램

```
[화면 업데이트]
    │
    └─► Zone별 카드 데이터 바인딩
        └─► 관제 정보 표시
```
