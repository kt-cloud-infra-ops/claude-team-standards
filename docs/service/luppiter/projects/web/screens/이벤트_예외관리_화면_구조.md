# Luppiter Web - 이벤트 예외 관리 화면 구조

## 개요

본 문서는 Luppiter Web의 **이벤트 예외 관리** 화면의 구조와 처리 흐름을 설명합니다.

---

## 화면 정보

- **URL**: `/view/evt/subEventExcpState`
- **JSP 파일**: `evt/subEventExcpState.jsp`
- **Controller**: `EvtController.java`
- **REST Controller**: `EvtExcpRestController.java`
- **설명**: 이벤트 예외 관리 화면

---

## 화면 구성 요소

### 1. 헤더 영역
- **제목**: 이벤트 예외 관리

### 2. 검색 필터 영역 (`search`)
- **예외기간**:
  - 시작일: `inSearchExcpStartDate`
  - 종료일: `inSearchExcpEndDate`
- **검색어**:
  - 검색 타입 선택: `kwSearchType` (예외제목, 이벤트명, 호스트명, IP)
  - 검색어 입력: `kwSearchName`
- **검색/초기화 버튼**: `btSearch`, 초기화 버튼

### 3. 그리드 영역
- **Toast UI Grid**: `grid`
- **페이지 크기 선택**: `slcPageSize` (20, 50, 75, 100, 200, 300, 전체)
- **액션 버튼**:
  - 예외 등록: `btnEventExcp`
- **페이징**: `divPaging`

### 4. 팝업
- **예외 수정/이력확인 팝업**: `test-popup1` (`subEventExcpStatePopup.jsp`)
- **이벤트 상세내역 팝업**: `test-popup2` (`subEventExcpStateDetailPopup.jsp`)
- **이벤트 예외 등록 팝업**: `test-excp-popup2` (`subEventStatePopupExcp.jsp`)
- **장비 추가 팝업**: `test-popup2-sub` (`subEventStatePopupDeviceAdd.jsp`)


### 2단계: 이벤트 예외 목록 조회

```
[조회 요청]
    │
    ├─► 검색 버튼 클릭 (btSearch)
    │   └─► 이벤트 예외 목록 조회 함수 호출
    │
    └─► 페이지 로드 시 자동 조회

    ▼

[API 호출]
    │
    └─► POST /api/evt/excp/list
        ├─► 파라미터 구성
        │   ├─► 예외기간 (시작일, 종료일)
        │   ├─► 검색 조건 (검색타입, 검색어)
        │   ├─► 페이지 정보 (pageNum, pageSize)
        │   └─► 정렬 정보

        └─► 응답 데이터
            ├─► 이벤트 예외 목록 (list)
            ├─► 전체 건수 (totalCount)
            └─► 페이징 정보

    ▼

[데이터 바인딩]
    │
    └─► 그리드 데이터 설정
        ├─► gridObj["grid"].resetData(data.list)
        └─► 페이징 정보 업데이트
```

#### 주요 코드 위치
- **조회 함수**: `subEventExcpState.jsp:이벤트 예외 목록 조회 함수`

## 필요한 데이터 (API 요청)

### 이벤트 예외 목록 조회 파라미터

| 필드명 | 타입 | 설명 | 필수 여부 |
|--------|------|------|-----------|
| `pageNum` | Integer | 페이지 번호 | 필수 |
| `pageSize` | Integer | 페이지 크기 | 필수 |
| `inSearchExcpStartDate` | String | 예외 시작일 | 선택 |
| `inSearchExcpEndDate` | String | 예외 종료일 | 선택 |
| `kwSearchType` | String | 검색 타입 (title/evtName/hostname/ip) | 선택 |
| `kwSearchName` | String | 검색어 | 선택 |

### 이벤트 예외 등록 파라미터

| 필드명 | 타입 | 설명 | 필수 여부 |
|--------|------|------|-----------|
| `title` | String | 예외 제목 | 필수 |
| `startDate` | String | 예외 시작일 | 필수 |
| `startTime` | String | 예외 시작시간 | 필수 |
| `endDate` | String | 예외 종료일 | 필수 |
| `endTime` | String | 예외 종료시간 | 필수 |
| `reason` | String | 예외 사유 | 필수 |
| `details` | Array | 대상 장비 목록 | 필수 |
| `details[].triggerId` | String | 트리거 ID | 필수 |
| `details[].hostname` | String | 호스트명 | 필수 |
| `details[].ip` | String | IP | 필수 |
