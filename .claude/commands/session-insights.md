> **실행 모델**: `opus` — 이 스킬을 Task 도구로 위임 시 `model: "opus"`를 사용하세요.

세션 데이터를 분석하여 인사이트 대시보드를 생성합니다.

## 데이터 소스
- `~/.claude/projects/` - 프로젝트별 세션 기록 (*.jsonl)
- `~/.claude/todos/` - TODO 기록

## 분석 항목

### 1. 전체 통계
다음 bash 명령어로 세션 데이터를 수집하세요:

```bash
# 모든 프로젝트의 세션 파일 목록
find ~/.claude/projects -name "*.jsonl" -type f 2>/dev/null

# 각 세션 파일에서 timestamp 추출하여 분석
for f in ~/.claude/projects/*/*.jsonl; do
  echo "=== $f ==="
  cat "$f" 2>/dev/null | jq -r 'select(.type=="user") | .timestamp' 2>/dev/null | head -5
done
```

### 2. 날짜별 세션 수
각 프로젝트의 jsonl 파일에서 timestamp를 추출하여 날짜별로 그룹핑:
- 최근 7일간 일별 세션 수
- 가장 활발했던 날

### 3. 시간대별 활동 패턴
timestamp의 시간대를 분석하여:
- 주로 작업하는 시간대 (아침/오후/저녁/심야)
- 시간대별 세션 분포

### 4. 프로젝트별 활동량
`~/.claude/projects/` 하위 폴더별로:
- 세션 파일 크기 (대략적인 활동량)
- 마지막 세션 시간
- 총 세션 수

### 5. 최근 세션 요약
가장 최근 세션들의:
- 첫 메시지 내용 (무슨 작업을 했는지)
- 세션 지속 시간 (첫 timestamp ~ 마지막 timestamp)
